N1C Gene Registry Info

1. How to navigate the registry

N-of-1 Projects
This part of the registry lists ongoing n-of-1 and n-of-few projects by N1C members as well as collaborators and further organizations. The projects are listed alphabetically by gene name. Each project can be opened as its own entry, containing additional information on the variant, status of the project, availability of Investigational New Drug Application (IND), and cell lines/animal models. Additionally, the entry pages also contain contact information for the lab and PI to reach out to. The search function will search the full registry, including parts of the entries that are not visible in the main table.
Marketed Genetic Therapies
This registry contains the Marketed Genetic Therapies available for larger groups of patients. The overview table is sorted alphabetically by marketed drug name. No single entries are available here. The search function will search through all marketed therapies.
Assessed Variants
This registry contains variants assessed for their eligibility for a therapeutic intervention. The variants are listed alphabetically by gene. Each variant entry can be opened to receive more information on the assessment. We indicate who performed the assessment of each variant. Even if a variant is eligible for a genetic therapy, that does not guarantee that a therapy can be developed. Assessments are made on the molecular level.
Published Genetic Therapies
This registry contains a list of published genetic therapies across different modalities.
Assessed Genes and Diseases
This registry contains genes and diseases that have been assessed using the N1C framework for assessing eligibility of genes and diseases for genetic intervention.
Automated Assessment Tools
Different tools are available that have automated the N1C variant assessment guidelines. We benchmark these tools using a manually curated dataset and provide results and standardized reports via our Gene Registry.

2. Access our data

The N1C Gene Registry data is free to use by academic and non-profit institutions. Please reach out to generegistry@n1collaborative.org for API access.

For-profits can contact the N1C to discuss options to access the registry data.
