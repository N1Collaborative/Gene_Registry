N1C Gene Registry API

This API provides programmatic access to both N1C registries. Data is freely available for non-commercial and academic use.

Available Endpoints

  • GET /api/data?table=N1C_projects
    Returns all data from the N1C Projects table.
    Try it
  • GET /api/data?table=marketed_drugs
    Returns all data from the Marketed Drugs table.
    Try it
  • GET /api/data?table=assessed_variants
    Returns all data from the Assessed Variants table.
    Try it
  • GET /api/search?q=ATM
    Searches all registries for entries that match ATM.
    Try it

Example Usage (JavaScript)

    fetch("https://gene-registry.onrender.com/api/data?table=N1C_projects")
      .then(res => res.json())
      .then(data => console.log("N1C Projects:", data));

  fetch("https://gene-registry.onrender.com/api/data?table=marketed_drugs")
    .then(res => res.json())
    .then(data => console.log("Marketed Drugs:", data));

  fetch("https://gene-registry.onrender.com/api/data?table=assessed_variants")
    .then(res => res.json())
    .then(data => console.log("Assessed Variants:", data));

  fetch("https://gene-registry.onrender.com/api/search?q=ATM")
    .then(res => res.json())
    .then(data => console.log("Search Results:", data));

Linking to Entries

You can link directly to specific entries in:

  • N1C Projects:
    https://generegistry.n1collaborative.org/entry.html?id=123
  • Assessed Variants:
    https://generegistry.n1collaborative.org/variant_entry.html?id=123
