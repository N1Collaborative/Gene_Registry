N1C Gene Registry API

This API provides programmatic access to the N1C Gene Registry. All data is freely accessible for non-commercial and academic use.

Endpoints

  • GET /api/data?table=N1C_projects
    Returns the full dataset of N1C-submitted projects.
    Try it
  • GET /api/data?table=marketed_drugs
    Returns the full dataset of marketed drugs.
    Try it

Example Usage (JavaScript)

fetch("https://gene-registry.onrender.com/api/search?q=ATM")
  .then(res => res.json())
  .then(data => console.log(data));

Linking to Entries

Each entry has an ID. You can link directly to an entry like this:

https://n1collaborative.github.io/Gene_Registry/entry.html?id=123

Terms of Use

This API is available for research and nonprofit purposes. Please cite the N1C Gene Registry if using this data in a publication or project.
