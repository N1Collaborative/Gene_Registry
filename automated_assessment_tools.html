About this section

This section is being developed to independently benchmark automated tools for therapeutic variant assessment against a protected manually curated dataset generated by trained N1C assessors.

Companies and developers can submit tools for thorough evaluation, including benchmarking against a dataset of manually assessed variants that is not publicly released and cannot be used for training. For each tool, we will provide a standardized report and publish the results openly through the N1C Gene Registry so that everyone can assess how well the tool performs.

Even if a tool is commercial or behind a paywall, the benchmarking results will remain freely accessible. Developers can then improve their tool and resubmit it for reassessment, allowing performance to be tracked transparently over time.

VariantMatcher AI

VariantMatcher AI is an automated tool designed to support therapeutic eligibility assessment for rare genetic variants by predicting suitable interventional genomic approaches based on variant and gene characteristics.

Standardized benchmarking report

Version tested v1.4
Date assessed April 2026
Benchmark dataset 200 protected variants
Overall agreement 84%
  • Strong performance on variants with clear therapeutic modality matches.
  • Reduced performance on edge cases requiring nuanced manual interpretation.
  • Good consistency across well-characterized genes.
  • Occasional overcalling of eligibility in more ambiguous scenarios.
Benchmarking was performed against an independent manually assessed dataset curated by trained N1C assessors. The underlying benchmarking dataset is not publicly released and cannot be used for model training.

GeneRoute

GeneRoute is an automated route-to-therapy prioritization tool that integrates variant interpretation with therapeutic pathway assessment.

Standardized benchmarking report

Version tested v2.1
Date assessed April 2026
Correct classifications 167/200
Overall agreement 83.5%
  • Good overall alignment with manual assessor consensus.
  • Performed particularly well for clearly defined variant classes.
  • More limited performance for variants requiring contextual interpretation beyond variant-level features.
  • Would benefit from refinement in complex eligibility scenarios.
Benchmarking results are freely accessible, regardless of whether the tool itself is publicly available or commercial.
