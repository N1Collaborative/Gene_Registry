Automated Eligibility Assessment Tools

This section is currently under construction.

About this section

This section is being developed to independently benchmark automated assessment tools for therapeutic actionability assessments against a protected manually curated dataset generated by trained N1C assessors.

Developers from academia and industry can submit tools for thorough evaluation, including benchmarking against a dataset of manually assessed variants. For each tool, we will provide a standardized report and publish the results openly through the N1C Gene Registry.

Tool A

Tool A is an automated tool designed to support therapeutic eligibility assessment for rare genetic variants by predicting suitable interventional genomic approaches based on variant and gene characteristics.

Standardized benchmarking report

Version tested v1.4
Date assessed April 2026
Benchmark dataset 200 protected variants
Accurcacy 84%
  • Strong performance on variants with clear therapeutic modality matches.
  • Reduced performance on edge cases requiring nuanced manual interpretation.
  • Good consistency across well-characterized genes.
  • Occasional overcalling of eligibility in more ambiguous scenarios.
Benchmarking was performed against an independent manually assessed dataset curated by trained N1C assessors. The underlying benchmarking dataset is not publicly released and cannot be used for model training.
